# Book Tracker (Трекер прочитанных книг)
Автор: Амиров Ринат

## Описание программы
«Book Tracker» — это графическое приложение (GUI) на Python, созданное с использованием библиотеки `tkinter`. Оно предназначено для удобного ведения списка прочитанных книг. Пользователи могут добавлять информацию о книгах, просматривать их в таблице, а также фильтровать список по жанру и объему (более 200 страниц). Данные автоматически сохраняются в формате JSON.

## Основные функции
- **Добавление книг**: поля «Название», «Автор», «Жанр» и «Количество страниц».
- **Валидация**: проверка на пустые поля и корректность ввода количества страниц (только положительные числа).
- **Таблица**: отображение всех записей в структурированном виде.
- **Фильтрация**:
    - Поиск по жанру (частичное совпадение).
    - Быстрый фильтр книг объемом более 200 страниц.
- **Сохранение данных**: автоматическое сохранение и загрузка из файла `data.json`.

## Инструкция по запуску
1. Убедитесь, что у вас установлен Python 3.x.
2. Скачайте файлы проекта.
3. Откройте терминал/командную строку в папке проекта.
4. Запустите программу командой:
   ```bash
   python main.py
   ```

## Примеры использования
- Чтобы добавить книгу, введите данные в соответствующие поля и нажмите «Добавить книгу».
- Чтобы найти все детективы, введите «детектив» в поле фильтра и нажмите «Применить фильтр».
- Чтобы увидеть только толстые книги, поставьте галочку «Только более 200 страниц» и нажмите «Применить фильтр».
