# GitHub User Finder (Поиск пользователей GitHub)
Автор: Амиров Ринат

## Описание программы
«GitHub User Finder» — это графическое приложение на Python, которое позволяет находить информацию о пользователях GitHub в реальном времени. Программа использует официальное GitHub API для получения данных и предоставляет возможность сохранять любимых разработчиков в список избранного.

## Основные функции
- **Поиск**: нахождение пользователя по его логину на GitHub.
- **Отображение данных**: показ логина, имени, количества публичных репозиториев и краткой биографии.
- **Избранное**: возможность добавить пользователя в локальный список для быстрого доступа.
- **JSON**: автоматическое сохранение и загрузка избранных пользователей из файла `favorites.json`.
- **Валидация**: проверка на пустое поле ввода перед началом поиска.

## Инструкция по запуску
1. Убедитесь, что у вас установлен Python 3.x.
2. Проверьте подключение к интернету.
3. Запустите приложение командой:
   ```bash
   python main.py
   ```

## Как использовать API
Программа использует публичную ветку GitHub API (`https://api.github.com/users/{username}`). 
- Для базовых запросов (до 60 в час) API-ключ не требуется.
- Если вы превысите лимит, GitHub может временно заблокировать ваш IP для запросов к API.

## Примеры использования
1. Введите в поле поиска логин, например `s-em1l`.
2. Нажмите кнопку «Найти».
3. Просмотрите информацию о пользователе.
4. Нажмите «В избранное», чтобы сохранить пользователя в таблице ниже.
