# Currency Converter (Конвертер валют)
Автор: Амиров Ринат

## Описание программы
«Currency Converter» — это графическое приложение на Python, разработанное с использованием библиотеки `tkinter`. Программа позволяет конвертировать валюты в режиме реального времени, используя данные внешнего API. Приложение сохраняет историю всех операций для последующего просмотра.

## Основные функции
- **Конвертация**: поддержка основных мировых валют (USD, EUR, RUB и др.).
- **Внешнее API**: получение актуальных курсов обмена через `open.er-api.com`.
- **История**: автоматическая запись всех конвертаций в таблицу.
- **JSON**: сохранение и загрузка истории из файла `history.json`.
- **Валидация**: проверка ввода суммы на корректность (положительное число).

## Инструкция по запуску
1. Убедитесь, что у вас установлен Python 3.x.
2. Проверьте наличие интернет-соединения (необходимо для получения курсов).
3. Запустите программу:
   ```bash
   python main.py
   ```

## API-ключ
В данной версии используется публичное API `open.er-api.com`, которое не требует регистрации API-ключа для базовых нужд. Если вы захотите использовать другое API (например, `exchangerate-api.com`), вам потребуется:
1. Зарегистрироваться на сайте.
2. Получить бесплатный API-ключ.
3. Вставить его в URL-запрос в коде программы.

## Примеры использования
- Введите сумму (например, `100`), выберите «USD» в поле «Из» и «RUB» в поле «В». Нажмите «Конвертировать».
- Просмотрите результат в нижней части формы и увидите запись в таблице истории.
